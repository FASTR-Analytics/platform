## Application Workflow

The typical workflow through the application:

```
1. Instance Setup
   ↓
2. Import Structure (admin areas & facilities)
   ↓
3. Import Health Data (HMIS/HFA)
   ↓
4. Create Project
   ↓
5. Configure Project Data Window
   ↓
6. Enable Modules
   ↓
7. Create Visualizations
   ↓
8. Build Reports
   ↓
9. Export and Share
```

Each step is covered in detail in the corresponding user guide.

## Key Features

### Multi-Level Organization
- **Instance**: Organization-wide workspace
- **Projects**: Focused analysis environments
- **Hierarchical structure**: Admin areas and facilities

### Flexible Data Management
- Import from multiple sources (HMIS, HFA, DHIS2)
- Version control for datasets
- Data windowing for focused analysis

### Powerful Analysis
- R-based analytical modules
- Module chaining for complex processing
- Automated execution and dependency management

### Rich Visualizations
- Charts, maps, and tables
- Flexible filtering and disaggregation
- Export capabilities

### Professional Reporting
- Multi-page reports
- PowerPoint and PDF export
- Customizable layouts

### Collaboration
- Role-based access control
- Project-level user management
- Viewer, editor, and admin roles
